# Kognis CLI Commands

> **Purpose:** Document the Kognis CLI commands as they are built
> **Stability:** EVOLVING

---

Commands will be documented here as they are implemented in the core daemon.

## Planned Commands

| Command | Description | Milestone |
|---|---|---|
| `kognis start` | Start the core daemon | M-025 |
| `kognis stop` | Graceful shutdown | M-025 |
| `kognis status` | Show daemon and plugin status | M-024 |
| `kognis plugin list` | List registered plugins | M-013 |
| `kognis plugin info <id>` | Show plugin details | M-013 |
| `kognis pipeline list` | List loaded pipelines | M-018 |
| `kognis pipeline show <id>` | Show pipeline slots and plugins | M-018 |
| `kognis health` | Show health aggregation | M-022 |
| `kognis dev` | Start in development mode with debug output | Future |
| `kognis plugin create <name>` | Scaffold a new plugin | Future |

---

*Commands are added to this document as they are implemented. Do not document planned commands as if they exist.*
